# Compare Command

Systematic comparison of items or concepts.

## Usage
`/compare "item1" "item2" ["item3"...]`

## Example
`/compare "Python" "JavaScript" "Go"`

## Output
- Feature comparison matrix
- Strengths and weaknesses
- Use case recommendations
- Summary table
